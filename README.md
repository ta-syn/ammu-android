# আম্মু অ্যাপ (Ammu App)

বাংলাদেশি মায়েদের জন্য একটি সম্পূর্ণ AI সহকারী অ্যাপ। 
নামাজের সময়, কোরআন, হাদিস, স্বাস্থ্য তথ্য, এবং AI চ্যাট — সবকিছু এখন বাংলায়।

## বৈশিষ্ট্যসমূহ
- **AI চ্যাট**: মায়েদের জন্য বিশেষায়িত ইসলামিক ও সাধারণ সহায়ক।
- **নামাজ ও কুরআন**: নামাজের সময়সূচী, কিবলা, তাসবিহ এবং কুরআন।
- **স্বাস্থ্য ও ঔষধ**: ঔষধের রিমাইন্ডার এবং স্বাস্থ্য ট্র্যাকিং।
- **অতিরিক্ত**: আবহাওয়া, খবর, রেসিপি, খরচ হিসাব এবং আরও অনেক কিছু।

## প্রজেক্ট সেটআপ
- এটি একটি Android Jetpack Compose অ্যাপ।
- API Keys (যেমন OpenRouter) `.env` ফাইলে সেট করতে হবে।
- ডেটাবেজ হিসেবে Room (Local SQLite) ব্যবহৃত হয়েছে।

## সিকিউরিটি ও প্রাইভেসি 
- সমস্ত সংবেদনশীল ডেটা ডিভাইসে লোকাল ডিরেক্টরিতে সেভ থাকে। 
- নেটওয়ার্ক ট্রাফিক এনক্রিপ্টেড। 

## রিলিজ 
- অ্যাপটি প্রোডাকশন-রেডি। 
- লাইটহাউস/Web Core Vitals এর বদলে Android Vitals এবং UI Performance অপ্টিমাইজ করা হয়েছে।
